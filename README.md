# 复习01 题库练习网页

这是一个纯静态题库练习网页，适合直接部署到 GitHub Pages。

## 文件说明

- `index.html`：首页配置、考试页、成绩报告弹窗
- `styles.css`：参考图风格的考试页、首页配置面板、报告弹窗样式
- `app.js`：自定义练习、错题本、保存、提交、计时、导入导出等交互
- `question-bank.json`：题库数据
- `.nojekyll`：让 GitHub Pages 原样托管静态文件

## 本地预览

由于页面使用 `fetch` 读取 `question-bank.json`，不要直接双击打开 HTML。请在本目录启动一个静态服务器：

```powershell
python -m http.server 8080
```

然后访问：

```text
http://localhost:8080
```

## GitHub Pages 部署

1. 把本文件夹内的所有文件上传到 GitHub 仓库根目录。
2. 打开仓库 `Settings`。
3. 进入 `Pages`。
4. `Build and deployment` 选择 `Deploy from a branch`。
5. 选择 `main` 分支和 `/root` 目录。
6. 保存后等待 GitHub 生成静态网页域名。

## 功能

- 首页由用户选择练习入口：全部题库、错题复习、未做优先、快速模式。
- 题量快捷按钮：10题、20题、50题、全部。
- 题型可选：单选题、多选题、判断题、简答题。
- 顺序可选：固定顺序、随机顺序、错题优先、未做优先。
- 快速模式一次只显示一题，答对自动进入下一题，答错停留；可查看答案，也可手动上一题/下一题。
- 右侧答题卡按题型分组，点击题号丝滑跳转。
- 选项点击后自动保存到当前浏览器。
- 提交后生成成绩报告：分数、正确率、错题、用时、未做题。
- 错题本自动收录答错题，答对一次自动移出，也支持手动加入/移除。
- 支持搜索题干、选项、答案。
- 支持导出、导入、清空本地进度。
